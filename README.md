# sluchawka `1.0.0-alpha`

**Sluchawka** is a music related content and sharing platform build with TypeScript and PostgreSQL.

## Dependencies

### Frontend

- React
- Next.js
- Chakra-UI
- Urql
- Storybook

### Backend

- Express
- TypeGraphQL
- Prisma
- PostgreSQL
- Apollo GraphQL
- Redis

## Features

- Browse songs, albums, artist
- Browse user created content like playlists, collections, etc.
- Music player
- Friends
- User profiles
- Blogs
